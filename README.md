# c6 – Обычный калькулятор

## Деплой на Vercel

1. Создайте репозиторий на GitHub/GitLab и загрузите этот проект.
2. Перейдите на [vercel.com](https://vercel.com) и импортируйте ваш репозиторий.
3. **Важно:** В настройках проекта Vercel (Settings → General → Build & Development Settings):
   - **Build Command**: Включите "Override" и оставьте поле **пустым** (удалите `npm run build`)
   - **Output Directory**: Включите "Override" и укажите **`.`** (точка) или оставьте пустым
   - **Install Command**: Включите "Override" и оставьте поле **пустым** (удалите `npm install`)
   - **Development Command**: Можно оставить как есть (None)
4. Сохраните настройки и задеплойте проект заново.

## Структура проекта
- `index.html` — основной файл калькулятора (должен быть в корне репозитория)
- `.gitignore` — игнорируемые файлы

## Локальный запуск
Откройте `index.html` в браузере.

---

_Для статического HTML-сайта файл `vercel.json` не требуется — Vercel автоматически определит проект как статический._
