# c6 – Простой калькулятор

## Деплой на Vercel

1. Создайте репозиторий на GitHub/GitLab и загрузите этот проект.
2. Перейдите на [vercel.com](https://vercel.com) и импортируйте ваш репозиторий.
3. Vercel автоматически определит, что это статический проект, и развернёт его. Главная страница – `index.html`.

## Структура проекта
- `index.html` — основной файл калькулятора
- `vercel.json` — конфигурация для деплоя
- `.gitignore` — игнорируемые файлы

## Локальный запуск
Откройте `index.html` в браузере.

---

_Для новой логики или расширения интерфейса – рекомендуем перенести исходники в папку `public/` или применить фреймворки (Next.js, React и пр.) при необходимости._
